# Revision Playbook

Use this reference for substantial writing tasks. Apply only the sections that match the user's request.

## Diagnosis Dimensions

1. **Claim**: Is the main point stated early and directly?
2. **Evidence**: Are facts, examples, data, methods, or observations present and verifiable?
3. **Logic**: Does each paragraph move from claim to support to explanation to implication?
4. **Specificity**: Are abstract labels replaced with concrete actors, actions, objects, and mechanisms?
5. **Register**: Does the prose match the expected genre and audience?
6. **AI Texture**: Does the text rely on formulaic transitions, symmetrical lists, empty intensifiers, or generic conclusion language?

## Common Fixes

### Direct Argument

Before:

```text
这并不是一种简单的工具应用，而是一种对研究流程的重新组织。
```

After:

```text
这种方法重新组织了研究流程，使资料筛选、论证搭建和结果解释可以在同一框架中推进。
```

### Evidence Support

Weak:

```text
该方法具有重要意义，并能显著提升研究质量。
```

Stronger:

```text
该方法把文献筛选标准、编码过程和结论解释放在同一流程中，减少了研究者在不同工具之间反复转换造成的遗漏。
```

If the user's material lacks evidence, write:

```text
该方法可能提升研究质量，但仍需要补充评价指标或案例结果来支撑这一判断。[需要证据]
```

### Paragraph Logic

Prefer:

```text
Claim -> Evidence -> Explanation -> Implication
```

Avoid:

```text
Background -> vague importance -> broad contrast -> claim at the end
```

### AI Trace Cleanup

Replace:

- "首先 / 其次 / 最后 / 综上所述" when they create a mechanical list.
- "具有重要意义 / 提供了新的视角 / 具有一定参考价值" when no specific contribution follows.
- Repeated three-part parallel sentences that sound templated.

Use:

- Natural transitions based on causality, chronology, contrast, evidence, or implication.
- Specific nouns and verbs.
- Sentence lengths that follow the idea, not a decorative rhythm.

## Genre Guidance

### Academic Abstract

Use four sentences by default:

1. Background and knowledge gap.
2. What the study did.
3. Core finding.
4. Implication.

The core finding should be the clearest sentence.

### Introduction or Proposal

Use:

1. Field problem.
2. Specific gap.
3. Why the gap matters.
4. What this work does.
5. Expected contribution.

Avoid opening with broad social value unless the user needs a policy-facing style.

### Analytical Report

Use:

1. Answer first.
2. Key evidence.
3. Drivers or mechanisms.
4. Risks, caveats, and next action.

### General Essay

Use:

1. Thesis.
2. Reason.
3. Example.
4. Interpretation.
5. Return to thesis.

## Output Examples

For a normal rewrite:

```text
Revised:
...

What changed:
- Put the positive claim first.
- Replaced broad value language with a concrete mechanism.
- Kept the caveat but moved it after the claim.
```

For diagnosis:

```text
Top issues:
1. The main claim appears too late.
2. The paragraph uses value labels without evidence.
3. The transitions sound formulaic.

Revision priorities:
1. State the claim in the first sentence.
2. Add one concrete example or method detail.
3. Replace list-like connectors with causal transitions.
```
